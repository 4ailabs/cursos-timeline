## BLOQUE 2 — La ciencia · 10 min

[Cambio de tono. Más tranquilo. Más preciso.]

Voy a contarte qué está pasando exactamente en tu cuerpo / cuando eso sucede. //

Porque durante muchos años / la medicina trabajó con músculos, / con huesos, / con órganos. /
Pero había un tejido que quedaba fuera de esa conversación. //

Se llama **fascia.** //

La fascia es el tejido conectivo que rodea absolutamente todo en tu cuerpo. /
Tus músculos. / Tus órganos. / Tus nervios. / Tus vasos sanguíneos. /
Tu cara. / Tu cuello. / Tu mandíbula. / Tu cráneo. //

Imagínalo como una red. / Una red que lo sostiene todo / y que al mismo tiempo lo conecta todo. //

Cuando esa red está hidratada, / fluida, / con movimiento — /
la sangre circula bien. /
La linfa drena. /
El sistema nervioso recibe las señales que necesita. //

Pero cuando esa red se endurece — /
y se endurece con el estrés, / con el apriete, / con la postura, / con los años — /
**todo lo que debería fluir / se detiene.** //

[Pausa.]

Y aquí está la clave. //

La fascia no envejece sola. /
La fascia **se endurece por falta de movimiento.** //

No es tiempo lo que la endurece. / Es inmovilidad. //

¿Qué significa eso? /
Que si le devuelves el movimiento, / le devuelves la función. //

Eso es exactamente lo que vamos a hacer esta noche. //

[Pausa.]

Ahora la ciencia concreta. //

En 2017, / un estudio publicado en los Institutos Nacionales de Salud de Estados Unidos / —el NIH— /
demostró que la manipulación mecánica de la piel — /
es decir, / trabajar la fascia con las manos — /
aumenta la producción de **procolágeno-1, / fibrilina / y tropoelastina.** //

Esas son las proteínas estructurales que le dan firmeza y elasticidad a tu piel. /
Las mismas que bajan con los años. /
Las mismas que cuestan una fortuna en sueros y tratamientos. //

El estudio mostró que el movimiento mecánico — / tus manos sobre tu fascia — /
activa los fibroblastos. /
Los fibroblastos son las células que fabrican colágeno. //

No es una crema. / No es una inyección. /
**Son tus propias manos sobre tu propio tejido.** //

[Pausa.]

Hay algo más que encontré / que me parece igual de importante. //

Un estudio publicado en Neuroscience News / demostró /
que la estimulación de la cara activa el **sistema glinfático** del cerebro. //

El sistema glinfático / es el mecanismo que usa el cerebro para limpiarse. /
Para eliminar los desechos metabólicos que se acumulan durante el día. /
Para consolidar la memoria. /
Para funcionar con claridad al día siguiente. //

Este sistema trabaja principalmente de noche — / mientras dormimos. /
Pero se activa / cuando hay movimiento en la fascia craneal. //

Esto explica algo que muchas personas reportan después de practicar Crania regularmente: /
que duermen diferente. /
Que amanecen con menos niebla. /
Que la cabeza se siente más despejada. //

No es placebo. / Hay un mecanismo detrás. //

[Pausa.]

Y una última conexión que quiero que veas — / porque esta es la que más sorprende. //

La fascia de tu cara / no trabaja aislada. /
Está conectada en líneas continuas / con el resto del cuerpo. //

La mandíbula / está en la misma línea fascial / que las caderas. /
Cuando liberas la mandíbula, / algo se suelta también en la pelvis. //

El cuello / tiene conexión fascial directa con el hígado. /
El hígado en la medicina tradicional — y en la fisiología moderna también — /
es uno de los órganos donde más se acumula la tensión crónica. //

Los ojos / están en la misma red que la vejiga. /
La vejiga es el órgano que más rápido responde / a los estados de alerta y de miedo. //

Cuando la fascia se suelta en un lugar, / el cuerpo responde en otro. /
**Lo van a sentir esta noche.** //

[Pausa larga.]

La fascia envejecida no es fascia vieja. /
Es fascia **sin movimiento.** //

Cuando le devuelves el movimiento, / le devuelves el flujo. /
Y donde hay flujo, / **hay colágeno. / Hay claridad. / Hay regulación.** //

## BLOQUE 5 — Ejercicio en vivo: Liberación de Ojos y Senos · 10 min

[Tono calmado. Voz clara. Sin prisa en ningún momento.]

Vamos a trabajar juntos ahora. //

Necesito que te acomodes. /
Si estás en una silla, / pon los pies en el suelo. / Los dos. / Bien apoyados. /
Afloja los hombros. / Suéltalos hacia abajo. //

[Pausa de 5 segundos.]

Abre la boca un momento. / Ciérrala sin apretar. /
Deja la mandíbula suelta. / Sin tensión. //

[Pausa.]

Esta técnica se llama **Liberación de Ojos y Senos.** //

Antes de empezar / quiero que sepas qué está pasando en esta zona del cuerpo. //

Los ojos, / los pómulos, / los senos nasales / — toda esta área — /
está conectada en la red fascial / con la **vejiga.** /
Y la vejiga es uno de los órganos que más rápido responde al estrés. /
Al miedo. / A la alerta. //

Cuando llevamos días, / semanas, / meses en estado de alerta — /
esa tensión se queda guardada aquí. /
En esta zona del rostro. //

Cuando la liberas, / el cuerpo drena. //

Es probable que después del ejercicio / sientas que necesitas ir al baño. /
Eso no es coincidencia. / El cuerpo está soltando algo que tenía guardado. //

También es posible que el ojo que trabajes primero / se vea diferente al otro / cuando termines. /
Más abierto. / Más descansado. /
Es normal. / El otro lado lo hacemos después. //

[Pausa.]

Bien. / Empezamos. //

[acción: tomar posición, mostrarse en cámara]

Vamos con el lado derecho primero. //

Con el **dedo medio de la mano derecha** — / el del centro — /
lleva la yema del dedo / justo al costado de la nariz. /
Al lado derecho. / Debajo del ojo. /
Donde la nariz se une con el pómulo. //

¿Están ahí? //

[Pausa breve. Dar tiempo.]

Bien. / Ahora presionen **hacia adentro.** /
No empujan hacia arriba. / No hacia abajo. /
Hacia adentro. / Como si quisieran presionar el hueso que está detrás. //

Mantengan esa presión. / No la suelten. //

[Pausa de 3 segundos.]

Ahora, / manteniendo la presión, / comiencen a mover el dedo muy despacio /
en un círculo alrededor del ojo. /
Siguiendo el **hueso orbital.** / El borde del hueso. /
No el párpado. / El hueso. //

[acción: hacer el movimiento lentamente, que la cámara lo vea]

Despacio. / Sin prisa. //

Vayan subiendo por el lado interno del ojo. /
Cruzan por arriba del ojo, / hacia la ceja. /
Bajan por el lado externo. /
Y regresan. //

[Pausa mientras hacen el movimiento.]

Una vez más. / Más despacio todavía. //

[Pausa.]

Ahora, / antes de sacar el dedo, /
pongan la **palma de la mano** contra el pómulo. /
Toda la palma. / Y ruédenla suavemente hacia atrás — /
como si quisieran separar el pómulo del resto de la cara. //

Mantengan ahí. //

[Pausa de 5 segundos.]

Ahora respiramos. //

**Inhalan por la nariz...** //

[acción: inhalar]

**Exhalan por la boca.** //

[acción: exhalar]

Una vez más. / Inhalan... / Exhalan... //

Tres. / Cuatro. / Cinco. / Seis. //

[Pausa larga tras la última respiración.]

Bajen la mano. //

[Pausa de 5 segundos.]

Abran la boca un momento. /
Muevan la mandíbula a los lados. /
Sientan si hay más espacio. //

¿Sienten la diferencia entre el lado derecho / y el lado izquierdo de la cara? //

El lado que trabajamos / debería sentirse más abierto. /
Más ligero. / Más vivo. //

[Pausa.]

Eso es la fascia respondiendo. //

Ahora hacemos el lado izquierdo. //

Con el **dedo medio de la mano izquierda.** /
Al costado izquierdo de la nariz. / Debajo del ojo izquierdo. //

Presionen hacia adentro. //

Círculo alrededor del ojo izquierdo. /
Mismo recorrido. / Mismo ritmo. /
Despacio. //

[Pausa mientras hacen el movimiento.]

Palma izquierda sobre el pómulo izquierdo. /
Ruédenla hacia atrás. //

Mantengan. //

**Inhalan...** / **Exhalan...** //

Seis respiraciones. //

[Dar tiempo para las 6 respiraciones.]

Bajen la mano. //

[Pausa larga.]

Ahora / con las dos manos / presionen suavemente ambos lados al mismo tiempo. /
Los dos pómulos. / Las dos zonas. / //

Cierren los ojos. //

Respiren una vez — / lento. //

[acción: respirar]

Y suelten. //

[Pausa de 8 segundos en silencio.]

Abran los ojos. //

[Pausa.]

Lo que acaban de sentir — /
esa diferencia, / esa apertura, / ese espacio que no estaba antes — /

**eso es una de quince técnicas.** //

## BLOQUE 6 — Lo que puedes ganar · 5 min

[Tono tranquilo. Dejar que el cuerpo procese antes de hablar.]

[Pausa.]

**La tensión facial y cervical se reduce.** /
No tienes que decidir soltarla. /
No tienes que "relajarte". /
Cuando le das movimiento al tejido, / se suelta solo. /

[Pausa.]

Y donde hay más flujo sanguíneo / hay más producción de colágeno. /
Eso se ve en el espejo. /
No de un día para otro. / Con el tiempo. /
Pero sin cremas. / Sin procedimientos. / Sin depender de nadie. //

[Pausa.]

**La linfa empieza a drenar.** /
Esa pesadez en la cara por las mañanas — / esa inflamación que ya se volvió normal — /
disminuye. /
Porque la linfa, a diferencia de la sangre, / no tiene bomba propia. /
Depende del movimiento. /

**El sistema nervioso se regula.** /
La fascia tiene una relación directa con el nervio vago. /
El nervio vago es el que regula el estado de calma del cuerpo. /
Cuando liberas la fascia craneal, / el vago responde. /
No porque alguien lo programe. /
