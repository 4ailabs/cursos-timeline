## 2. FUNDAMENTO TEÓRICO CENTRAL

### 2.1 ¿Qué es el Set Point?

El set point es un **rango biológico predeterminado** que el cuerpo defiende automáticamente. No es un número fijo sino una banda de operación que el organismo mantiene mediante retroalimentación constante.

**Tres características fundamentales:**
1. **Automático** — no requiere decisión consciente
2. **Inconsciente** — opera por debajo de la percepción
3. **Involuntario** — no se puede anular con fuerza de voluntad

### 2.2 Los 8 Sistemas Automáticos

El cuerpo defiende set points en 8 áreas:

| # | Sistema | Ejemplo |
|---|---------|---------|
| 1 | Temperatura corporal | ~36.5-37°C |
| 2 | pH sanguíneo | 7.35-7.45 |
| 3 | Presión arterial | Rango individual |
| 4 | Glucosa | 70-100 mg/dL |
| 5 | Dolor | Umbral individual |
| 6 | Emociones | Tono emocional basal |
| 7 | Peso corporal | Rango defendido |
| 8 | Sistema nervioso (PHS) | Tono simpático/parasimpático |

### 2.3 El Cerebro Triuno (MacLean)

Modelo jerárquico de poder cerebral:

| Cerebro | Función | Poder | Recurso |
|---------|---------|-------|---------|
| **Reptiliano** (tallo) | Supervivencia | MÁXIMO | Ilimitado |
| **Límbico** (emocional) | Emociones, memoria | Alto | Energía ilimitada |
| **Neocórtex** (racional) | Lógica, voluntad | MÍNIMO | 20-30 min máximo |

**Implicación clínica fundamental:** La fuerza de voluntad (neocórtex) se agota en 20-30 minutos de resistencia consciente. Después, el sistema límbico toma el control. Las dietas basadas en restricción fracasan porque dependen del eslabón más débil de la cadena cerebral.

### 2.4 Neuroception

Capacidad automática de detectar peligro sin procesamiento consciente. El cuerpo evalúa el entorno y responde antes de que la mente racional se active. Concepto crucial para entender por qué el peso se defiende sin que la persona "decida" mantenerlo.

## 3. NEUROBIOLOGÍA DEL SET POINT

### 3.1 Fuerza de Voluntad como Recurso Finito

- La voluntad consciente dura máximo 20-30 minutos de resistencia activa
- Después de ese período, el sistema límbico retoma el control
- Por eso todas las dietas basadas en restricción tienen una tasa de fracaso del **95-98% a 5 años** (estudio 2007)

### 3.2 Peso como Memoria Procedimental

El peso corporal se almacena como **memoria implícita**, igual que andar en bicicleta. No se puede "desaprender" con lógica; requiere reprocesamiento a nivel inconsciente.

### 3.3 Las 4 Etapas del Aprendizaje

1. **Incompetencia inconsciente** — no sé que no sé
2. **Incompetencia consciente** — sé que no sé
3. **Competencia consciente** — sé, pero requiere esfuerzo
4. **Competencia inconsciente** — sé sin pensar (nuevo set point)

El objetivo terapéutico es llevar un nuevo peso a la etapa 4: competencia inconsciente.

### 3.4 Neuroplasticidad

- Lo que se repite se consolida
- El cerebro aprende por repetición, no por moralidad
- No distingue "bueno" de "malo", solo "frecuente" de "infrecuente"
- Implicación: cambiar el set point requiere repetición sostenida, no fuerza de voluntad puntual

### 3.5 Las 3 Fases Defensivas al Cruzar el Límite Inferior del Set Point

Cuando el peso baja por debajo del set point establecido, el cuerpo activa defensas escalonadas:

| Fase | Pérdida | Mecanismo | Síntomas |
|------|---------|-----------|----------|
| **Fase 1: Alerta** | 5-10% | Leptina baja, grelina sube | Hambre creciente, pensar más en comida |
| **Fase 2: Emergencia** | 10-15% | Reducción metabólica 15-25% | Temperatura baja, fatiga, letargo |
| **Fase 3: Colapso sistémico** | >15% | Preservación total | Libido cae, sistema inmune baja, obsesión alimentaria |

### 3.6 Estudios Científicos Clave

- **Experimento de Inanición de Minnesota (1944):** 36 hombres, 1560 cal/día, 6 meses → al recuperar alimentación normal, ganaron +10% de grasa extra sobre su peso original
- **Fothergill 2016 (Biggest Loser):** Participantes después de 6 años quemaban 500 calorías menos que el promedio. El metabolismo NO se recuperó
- **Estudio ACE (Felitti, 1998):** 7 experiencias adversas en infancia; con 4+ ACEs el riesgo de obesidad aumenta +400%
- **Tasa de fracaso de dietas:** 95-98% a 5 años (estudio 2007)

## 4. MARCO CONCEPTUAL PROFUNDO

### 4.1 Homeostasis vs. Alostasis

- **Homeostasis:** Mantiene el set point actual; el cuerpo defiende su rango establecido
- **Alostasis:** Establecimiento de un NUEVO set point tras cambio significativo o sufrimiento prolongado
- El objetivo terapéutico: generar alostasis hacia un set point más saludable

### 4.2 Significados → Creencias → Certezas

Jerarquía del procesamiento de experiencia:
1. Un evento recibe un **significado** (interpretación)
2. El significado repetido se convierte en **creencia**
3. La creencia consolidada se vuelve **certeza** (opera como "verdad" biológica)

### 4.3 Resignificación: La Herramienta Más Poderosa

- Única herramienta voluntaria consciente para cambiar el set point
- Cambiar el significado cambia la respuesta biológica
- "El arma más poderosa" según el marco del curso
- No cambia el evento, cambia la interpretación → cambia la respuesta corporal

### 4.4 Comunicación Interhemisférica

La integración entre hemisferios cerebrales permite el reprocesamiento de experiencias. El Protocolo de Mariposa utiliza estimulación bilateral (tapping alternado) para facilitar esta integración.

### 4.5 Diálogo Dual

| Consciente | Inconsciente |
|------------|-------------|
| Palabras | Sensaciones |
| Lógica | Imágenes |
| Argumentos | Metáforas |
| "Debería" | "Siento" |

### 4.6 Escucha Somática Profunda

Alternativa a los tests psicológicos convencionales. Se basa en observar las respuestas corporales del paciente más que en sus explicaciones verbales. El cuerpo no miente; las palabras pueden engañar.

### 4.7 El Permiso

Pregunta clave antes de cualquier trabajo de set point:
> *"¿Te permitirías sentir bienestar?"*

Si la respuesta no es un sí claro, hay un conflicto activo que impide el cambio. Es el primer paso diagnóstico.

### 4.8 Agencia (Michael Levin)

Los organismos tienen intenciones y metas propias, no solo ejecutan tareas. El cuerpo no "está gordo por error"; el peso cumple una función biológica precisa.

### 4.9 Causalidad Psíquica

Detrás de cada movimiento biológico existe una intención. El set point de peso no es aleatorio; es la respuesta perfecta a un conflicto específico.

### 4.10 Metáfora de Actualización de Software

- El cuerpo es el hardware (no se puede cambiar)
- Las creencias/memorias son el software (se pueden actualizar)
- Analogía: "Estás corriendo Windows 98 en un mundo que ya va en Windows 11"
- No se destruye el software antiguo; se actualiza → resignificación

## 6. LOS 6 PATRONES DE PESO

### Patrón 1: Derecho a Existir
- **Manifestación:** Retención de agua/edema, sistema renal
- **Conflicto central:** Abandono, "no debería estar aquí"
- **Mecanismo:** Síndrome de abandono
- **Dato clave:** 27 creencias nucleares organizadas en 3 categorías

### Patrón 2: Escasez/Hambre
- **Manifestación:** Grasa acumulada, metabolismo lento
- **Conflicto central:** "Nunca hay suficiente"
- **Mecanismo:** Ecuación comida = afecto, fantasía catastrófica
- **Dato clave:** El cerebro no distingue hambre real de restricción voluntaria

### Patrón 3: Desorientación Existencial / Síndrome de la Oveja Perdida
- **Manifestación:** Grasa visceral, cortisol elevado
- **Conflicto central:** Pérdida de referentes vitales
- **Mecanismo:** Estrés crónico por falta de dirección

### Patrón 4: El Escudo
- **Manifestación:** Grasa estructural protectora
- **Conflicto central:** Devaluación, agresión recibida
- **Mecanismo:** El cuerpo crea armadura literal
- **Dato clave:** Esta grasa NO se quema con ejercicio convencional

### Patrón 5: La Paradoja / Camuflaje
- **Manifestación:** Ciclo de vergüenza, peso como disfraz
- **Conflicto central:** Lo que resistes se fortalece
- **Mecanismo:** Camuflaje post-abuso, invisibilización
- **Dato clave:** La obsesión por adelgazar genera más peso (paradoja)

### Patrón 6: Hipotiroidismo
- **Manifestación:** Reducción T3/T4, metabolismo lento
- **Conflicto central:** Miedo al futuro, "frenar el tiempo"
- **Mecanismo:** La tiroides desacelera todo
- **Dato clave:** Para nódulos tiroideos, considerar conflicto nido/gaviota

## 8. EPIGENÉTICA Y TRANSMISIÓN TRANSGENERACIONAL

### 8.1 Epigenética como "Actualización de Software"

- Cambios en la expresión génica SIN cambiar el ADN
- El ambiente modifica qué genes se activan o silencian
- Ejemplo: Hambruna holandesa (Dutch Famine) — descendientes de madres que sufrieron hambruna nacieron con tendencia genética a acumular grasa, 2-3 generaciones después

### 8.2 Hipótesis del Gen Ahorrador (Thrifty Gene)

Perspectiva evolutiva: los genes que favorecen la acumulación de grasa fueron ventajosos durante milenios de escasez. En un mundo de abundancia, ese mismo programa biológico genera obesidad.

### 8.3 Las 6 Ventanas Temporales

| Ventana | Período | Características |
|---------|---------|-----------------|
| **Transgeneracional** | Antes de concepción | Memorias heredadas, epigenética |
| **Gestacional** | Concepción-nacimiento | Proyecto/sentido, emociones maternas |
| **Infancia** | 0-7 años | Mandatos, imprinting, apego |
| **Pubertad** | Cambio hormonal | Identidad, sexualidad, pertenencia |
| **Adultez** | Eventos de vida | Conflictos activos, pérdidas |
| **Sistema familiar** | Atemporal | Lealtades invisibles, exclusiones |

### 8.4 Los 4 Espacios Temporales (para diagnóstico)

1. **Ahora** — 0 a 21 días (conflicto activo actual)
2. **Pasado** — 21 días hasta concepción (historia personal)
3. **Gestacional** — vida intrauterina
4. **Transgeneracional** — herencia ancestral

### 8.5 Proyecto/Sentido

- Mandatos recibidos antes de los 7 años (especialmente gestacionales y primeros 3 años)
- El cuerpo los obedece literalmente
- Ejemplo: "Debes ser grande y fuerte" → el cuerpo crece en volumen para cumplir

### 8.6 Trauma Simbiótico (Ruppert)

Si el niño concibe que la madre no lo ama, biológicamente equivale a morir. El cuerpo puede generar peso como mecanismo de supervivencia ante la percepción de falta de amor materno.

## 9. PERTENENCIA COMO SET POINT

### 9.1 Condicionamiento Social

La pertenencia al grupo es una necesidad biológica de supervivencia (Maslow). El peso puede ser un mecanismo de pertenencia:
- "En mi familia todos somos gordos" → lealtad biológica
- Adelgazar puede significar inconscientemente "traicionar al clan"
- El cuerpo defiende el peso del grupo al que pertenece

### 9.2 Conflictos Sistémicos

Dinámicas ocultas del sistema familiar:
- **Lealtades invisibles** — peso como tributo a un ancestro
- **Exclusiones** — llevar el peso del miembro rechazado
- **Roles no reconocidos** — parentificación, hijo de reemplazo
- **Identificaciones inconscientes** — "soy como el tío que murió obeso"

### 9.3 Parentificación

Inversión de roles donde el niño cuida emocionalmente al padre/madre. El volumen corporal crece para "tener la capacidad" de sostener a los padres.

## 10. EL ERROR DE LA RESTRICCIÓN

### 10.1 Mecanismo

El cerebro interpreta la restricción alimentaria voluntaria como hambruna:
- **Grelina sube** → hambre incontrolable
- **Metabolismo baja** → ahorro energético
- **Obsesión alimentaria** → el cerebro no piensa en otra cosa
- **Efecto rebote** → recupera peso + extra (comprobado en Minnesota y Biggest Loser)

### 10.2 Distinción Clave

| Concepto | Definición | Respuesta corporal |
|----------|-----------|-------------------|
| **Hambre** | Hay comida disponible, no como | Tolerable |
| **Ayuno** | Restricción planificada con fin | Adaptable |
| **Hambruna** | No hay comida | PÁNICO → acumulación máxima |

El cerebro interpreta las dietas restrictivas como hambruna, no como ayuno.

## 13. PROTOCOLO DE MARIPOSA (BUTTERFLY PROTOCOL)

### 13.1 Descripción General

Protocolo de reprocesamiento emocional y corporal basado en estimulación bilateral (tapping alternado). Se llama "de Mariposa" por la posición de las manos cruzadas sobre el pecho durante el tapping. Existen 33 versiones individualizadas, una para cada conflicto de obesidad.

### 13.2 Estructura de 5 Pasos (idéntica en los 33 protocolos)

**PASO 1: Activación de la Emoción (30 seg)**
- Tapping bilateral lento
- Repetición de frases activadoras del conflicto específico
- Objetivo: traer la emoción al presente

**PASO 2: Reconocimiento del Origen (60 seg)**
- Identificar momento/contexto de origen
- Observación corporal: ¿dónde se siente?
- Tapping continuo mientras se explora

**PASO 3: Verdad Transformadora (90 seg)**
- Introducir la verdad que resignifica el conflicto
- Tapping bilateral mientras se instala la nueva información
- "La guerra terminó" — mensaje central de actualización

**PASO 4: Instalación de Nueva Realidad (90 seg)**
- Tapping más rápido
- Frases de nuevas creencias
- Consolidación del nuevo significado

**PASO 5: Anclaje Corporal (60 seg)**
- Tapping más lento
- Anclar la nueva sensación en el cuerpo
- Registro somático del cambio

### 13.3 Elementos Post-Sesión (en cada protocolo)

- **Diario de reflexión:** preguntas guiadas específicas al conflicto
- **Escala de creencia:** 0-10 antes y después de cada sesión
- **Acción complementaria semanal:** tarea práctica para reforzar
- **Rituales simbólicos:** actos concretos de resignificación (ej: devolver simbólicamente la carga, carta al ancestro, ritual de diferenciación)
- **Contraindicaciones:** indicaciones de cuándo derivar a profesional especializado

### 13.4 Diferencia entre Parte 1 y Parte 2

| Aspecto | Parte 1 (1-12) | Parte 2 (13-33) |
|---------|----------------|-----------------|
| Tipo | Conflictos del momento | Conflictos de toda la vida |
| Inicio | Identificable | "Siempre ha sido así" |
| Repeticiones | Pocas sesiones | 10-20+ repeticiones |
| Profundidad | Evento específico | Patrón arraigado |

## 14. SECUENCIA TERAPÉUTICA CENTRAL

### 14.1 Principio: El Set Point como Solución Adaptativa

El peso NO es un error. Es la **solución perfecta** que el cuerpo encontró para un contexto específico. Antes de cambiar el set point, se debe reconocer y agradecer su función protectora.

### 14.2 Secuencia Validar → Agradecer → Actualizar

1. **Validar:** "Tu cuerpo hizo exactamente lo correcto en ese momento"
2. **Agradecer:** "Gracias por protegerme de [conflicto]"
3. **Actualizar:** "La guerra terminó. Ya no necesito esta protección"

### 14.3 Desensibilización Gradual

Cambiar el set point gradualmente para evitar activar los mecanismos defensivos (Fases 1-3). Cambios bruscos disparan la alarma biológica; cambios pequeños y sostenidos permiten la alostasis.

### 14.4 Justificación como Mecanismo de Supervivencia

Concepto de Hahnemann: el organismo justifica sus adaptaciones para mantener coherencia interna. Entender la justificación sin juzgarla es parte del proceso terapéutico.

### 14.5 Disociación Cognitiva

Mecanismo de supervivencia donde la persona separa lo que sabe de lo que siente. El terapeuta debe trabajar con ambos niveles simultáneamente (diálogo dual).

## 18. PRINCIPIOS RECTORES DEL CURSO

1. **El peso no es el problema; es la solución.** El conflicto subyacente es el problema real.
2. **La fuerza de voluntad es el recurso más débil.** No se puede vencer al cerebro reptiliano con lógica.
3. **Lo que se repite se consolida.** La repetición crea el nuevo set point, no la intensidad.
4. **Cambiar el significado cambia la biología.** La resignificación es la herramienta central.
5. **Validar antes de cambiar.** El cuerpo solo suelta su protección cuando se siente seguro.
6. **La guerra terminó.** Mensaje central que el inconsciente necesita recibir.
7. **Gradual, no radical.** Los cambios bruscos activan defensas; los graduales permiten alostasis.
8. **El cuerpo no miente.** La escucha somática supera a la explicación racional.
9. **Todo peso tiene intención.** No hay obesidad sin conflicto; no hay conflicto sin solución biológica.
10. **El terapeuta no cura; actualiza.** El papel es ayudar a actualizar el software, no destruirlo.
